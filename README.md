# maxwellcalkin

Personal site for Maxwell Calkin — AI alignment researcher, builder, father.

> Building intelligence in service of the widening and deepening of consciousness.

Static site, no build step. Open `index.html` locally or deploy the directory as-is to any static host.

## Files
- `index.html` — page markup
- `styles.css` — styling
- `main.js` — interactions and scroll behaviour
- `cosmos.js` — ambient background animation
